# Company Language

## Standard Company Description

Jera Technologies designs and builds practical AI enabled software products, automation systems, and enterprise modernization solutions.

## Short Description

Jera Technologies builds practical AI enabled software for clearer workflows, better decision support, and modernized systems.

## Service Description

Jera Technologies helps organizations design, modernize, and automate software workflows through applied AI, systems integration, and product focused engineering.

## Product Portfolio Description

Jera Technologies maintains a product portfolio of Jera built products that demonstrate its engineering approach. These products show how structured analysis, signal gathering, reviewable outputs, and productized workflows can become usable software experiences.

## Product Language

Use these phrases when describing products:

- Jera built product
- Product portfolio
- Product example
- Solution example
- Applied AI software
- Decision support application
- Structured analysis pipeline
- Reviewable decision artifact
- Productized AI workflow

## Claim Boundaries

Use language that is accurate and grounded. It is acceptable to say products were built by Jera Technologies. Do not describe a product as client work unless that is explicitly provided later.
